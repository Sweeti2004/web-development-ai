Project Title: "Friend Request"

Description: This HTML code creates a simple web page featuring a card layout. The card displays an image, a name ("Drama Queen"), and a status ("Stranger"). There's also a button labeled "Add Friend" which, when clicked, changes the status to "Friends" and changes the button text to "Remove". Clicking the button again changes the status back to "Stranger" and reverts the button text to "Add Friend".
